This repository contains 5 internship tasks completed for **Saiket Systems**.

### Task 1: Loan EMI Calculator
**File:** `LoanEMICalculator.java`
- Calculates monthly loan installments using EMI formula
- Interactive user input for principal, interest rate, and tenure
- Displays total amount and interest payable

### Task 2: Fetch Data from API
**File:** `FetchAPIData.java`
- Fetches data from public APIs (Jokes, Quotes, Users)
- Displays data in user-friendly format
- Uses HttpURLConnection for API calls

### Task 3: Simple Blog App
**File:** `BlogApp.html`
- Create, read, and delete blog posts
- Uses browser's local storage
- Modern responsive design

### Task 4: Number Guessing Game
**File:** `NumberGuessingGame.java`
- Guess a random number between 1-100
- Provides hints (too high/too low)
- Tracks attempts and shows statistics

### Task 5: Simple Contact Book
**File:** `ContactBook.java`
- Add, view, search, and delete contacts
- Stores name, phone, and email
- Simple menu-driven interface
